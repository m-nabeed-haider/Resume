# Cloud-Native DevSecOps Platform

## Overview

Semester project completed as part of the Cloud Computing course.

## Description

Developed a DevSecOps-oriented project by deploying a Netflix Clone on AWS using Amazon EKS. Built a CI/CD pipeline with Jenkins to automate application deployment and gain practical experience with cloud-native deployment workflows.

## Tech Stack

- AWS
- Amazon EKS
- Docker
- Jenkins
- CI/CD
