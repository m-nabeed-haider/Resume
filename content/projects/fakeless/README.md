# FakeLess

## Overview

FakeLess is my Final Year Project focused on defending against neural voice cloning attacks. The system learns a universal perturbation that modifies speech in a way that preserves human perceptual quality while significantly reducing the effectiveness of AI voice cloning models.

Instead of detecting deepfakes after they are generated, FakeLess aims to prevent malicious voice cloning before it happens.

---

## Project Type

Final Year Project (Team)

---

## Problem Statement

Modern voice cloning models can reproduce a person's voice from only a few seconds of speech.

FakeLess protects users by generating adversarial perturbations that degrade cloning performance while maintaining natural audio quality.
