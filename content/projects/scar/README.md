# SCAR

## Overview

SCAR is a modular multi-agent AI consulting platform designed to automate software consulting workflows using specialized AI agents.

The project explores modern agentic AI architectures by combining LangGraph, FastAPI, and locally hosted LLMs through Ollama.

## Features

- Multi-agent orchestration
- Modular architecture
- Local LLM support
- REST API
- Extensible workflow design

## Tech Stack

- Python
- LangGraph
- FastAPI
- Ollama
- Docker
