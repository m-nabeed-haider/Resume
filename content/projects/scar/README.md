# SCAR

## Overview

SCAR is a Jarvis-inspired multimodal AI assistant designed to combine conversational AI, Retrieval-Augmented Generation (RAG), image generation, and local system automation into a single extensible platform.

Unlike traditional chatbots, SCAR integrates external knowledge sources, live APIs, and locally hosted AI models to deliver context-aware and privacy-focused assistance.
