# ResumeLaTeX

A modular LaTeX resume system designed to generate multiple role-specific resumes from a shared content base.

## Features

- Custom resume class
- Modular content architecture
- Python Resume
- AI Resume
- MLOps Resume
- Automatic PDF generation
- GitHub Actions
- Versioned releases

## Build

make

## Outputs

output/
├── python_resume.pdf
├── ai_resume.pdf
└── mlops_resume.pdf
